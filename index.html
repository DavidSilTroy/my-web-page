Internship

Stock Management System at CERcuits

Between the period of 27/02/2023 to 26/05/2023, I completed a 13-week internship at CERcuits.

CERcuits is a company that specializes in ceramic printed circuit boards(PCB) and substrates for prototypes & small series which has experienced growth and needs to improve its inventory system. The current process of storing data is time consuming and inefficient, so the goal of the assignment was to create a system that provides better control of inventory and sends notifications based on specific parameters, such as low inventory or approaching due dates.

The need for an inventory management system stems from the fact that it is becoming increasingly difficult to manage inventory manually as the company expands its customer base, receives more orders and deals with a wider range of products. The system must answer questions such as what products are available, where they are located, what to order, when to order it and information about expiration dates.

The implementation of the inventory management system depends on an analysis of the information provided by the company about the required system, taking into account the pros and cons of different options that fit the requirements of the job.

Requirements

The assignment has several requirements that need to be fulfilled, although they are subject to change as further research is conducted.

The most important requirements are:

  • The use of open-source software is preferred.
  • The core application should run on a virtual machine with a Linux OS.
  • Select the frameworks and Relational Database.
  • Documentation with UML diagrams, database structure, and user stories.
  • Web-browser version of the Stock Management System.
  • Mobile version of the Stock Management System.
  • Documentation of the Developed Stock Management System.

What was expected

To fulfill the requirements of the Stock Management System, the company needs an automated system with user-friendly interfaces that provide accurate inventory information. This includes the option to scan QR codes to access stock information and update quantities. The system should be accessible through web browsers on computers and a dedicated Android application on mobile phones. Additionally, the system should replace the current stocking method and support the production department.

Arduino: Electric Surf table program

Using Arduino for an electric surf table. The program utilizes an analog signal sent to an Electronic Speed Controller (ESC) to control the motor. And With 5 buttons to have control of the surf table speed, having a timer for each button to save battery and prevent surf table is continue working if the user fall.

The Arduino, motor, ESC, and a LiPo battery work together to enable the following functionality:

  1. 100% Power for 8 seconds, then 75% Power for 60 seconds:
    • When the "100" button is pushed, the program sets the power output to 100% for a duration of 8 seconds.
    • After 8 seconds, the power output automatically shifts to 75% and maintains this level for 60 seconds.
  2. 75% Power for 100 seconds:
    • When the "75" button is pressed, the program sets the power output to 75% for a duration of 100 seconds.
    • After 100 seconds, the power output automatically shifts to 0%.
  3. 50% Power for 100 seconds:
    • When the "50" button is pressed, the program sets the power output to 50% for a duration of 100 seconds.
    • After 100 seconds, the power output automatically shifts to 0%.
  4. 25% Power for 100 seconds:
    • When the "25" button is pressed, the program sets the power output to 25% for a duration of 100 seconds.
    • After 100 seconds, the power output automatically shifts to 0%.
  5. Stop Button:
    • The "stop" button serves as an interrupt button, halting the execution of all other operations.
    • When the "stop" button is pressed, it cancels any ongoing power output or time intervals set by the other buttons.

Overall, this Arduino program provides a user interface with buttons that allow for controlling the power output of the electric surf table's motor. Each button press triggers a specific power level and duration, and the stop button can be used to immediately halt any ongoing operations.
